Arduino Xively multiple datastreams
===================================

Application that post multiple datastreams of sensor data from an Arduino to a Pachube/Cosm/Xively account.

Currently it has the following sensors:

1 Voltage sensor
4 current sensors: 
    source: http://www.pololu.com/catalog/product/1187
1 temperature sensor:
    source: http://datasheets.maximintegrated.com/en/ds/DS18B20.pdf
